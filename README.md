# Retro language support in Atom

Syntax highlighting and snippets for Retro Notes in Atom.

Format like markdown lists but with +, -, ? and ∆:

```
+ Positive
+ Positive

- Negative
- Negative

? Question
? Question

∆ Do Differently
∆ Do Differently
```

Get green, red, orange and blue syntax highlighting:
